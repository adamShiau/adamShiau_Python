Aegiverse AHRS Script

Author: Adam Shiau
Date: 2025-08-28
Version: 1.0

Requirement

pip install pyserial

Basic Usage

a. List available COM ports

python aegiverse_ahrs.py list

b. Start reading data

Note: After powering on, please wait 5 seconds before executing data reading.

Windows:

 python aegiverse_ahrs.py 1 COM27

Linux:

python aegiverse_ahrs.py 1 /dev/ttyUSB0

c. Stop reading data

c.1 Send stop command

Windows:

python aegiverse_ahrs.py 0 COM27

Linux:

python aegiverse_ahrs.py 0 /dev/ttyUSB0

c.2 Keyboard interrupt

Press Ctrl + C

Output Format

Each line corresponds to one packet. Values are comma-separated, without field names.

TIME(s),WX(°/s),WY(°/s),WZ(°/s),AX(m/s²),AY(m/s²),AZ(m/s²),PD_TEMP(°C),PITCH(°),ROLL(°),YAW(°)

Example:

5.364,0.017307,0.081714,0.003035,-0.032019,0.047499,9.744682,38.500,0.188685,0.204021,4.489222
